# Super Simple Notes 📝

Application mobile de prise de notes moderne, performante et sécurisée.

## 🚀 Fonctionnalités
- ✅ **Authentification Cloud** : Accès sécurisé à vos notes depuis n'importe quel appareil.
- ✅ **Gestion complète des notes** : Créer, modifier, supprimer et lire.
- ✅ **Catégories & Couleurs** : Organisation visuelle par tags.
- ✅ **Recherche intelligente** : Trouvez vos notes instantanément.
- ✅ **Mode Favoris** : Gardez vos notes importantes sous la main.
- ✅ **Synchronisation Temps Réel** : Vos modifications sont instantanées grâce à Firestore.

## 🛠 Stack Technique
- **Mobile** : React Native & Expo (TypeScript)
- **Design** : NativeWind (Tailwind CSS)
- **Database/Auth** : Firebase (Firestore & Auth)
- **State Management** : Zustand
